cs2
===

Combined simulator and computer target for KISS

Requirements
===========

* Qt 5.0 or higher
* CMake 2.8.12 or higher
* OpenCV 2.4.8
* libkovanserial
* pcompiler
* libkar
* libkovan

Building (OS X and Linux)
=========================

	cd cs2
	mkdir build
	cd build
	cmake ..
	make

You should now have cs2 ready to go in the deploy/ directory.
